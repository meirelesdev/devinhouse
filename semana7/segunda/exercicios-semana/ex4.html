Exercicio 4

A Promise a seguir executa um código que retorna um determinado valor. Identifique se esse valor é um erro (rejeição) ou um valor válido. 

                const promise = new Promise ( (callbackUm, callbackDois) => {
                    callbackDois(“Este valor é um resolve ou um reject???”)
                })

Resposta: Podemos verificar que o valor é um erro, pois a função que é executada em caso de erro é o segundo parâmetro da função que uma promisse recebe. O primeiro é a função executada quando a resposata for sucesso.
