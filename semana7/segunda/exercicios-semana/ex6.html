Exercicio 6

Crie uma função que retorne uma Promise que resolve após 3 segundos com o objeto abaixo. 

        const credentials = {
            username: “usuario”,
            email: “usuario@email.com”,
            authorized: true
         }

Resposta: O retorno da função pode ser observado no DevTools 

        function getAuth() {
            return new Promise((resolve) => {
                setTimeout(() => {
                    resolve({username: "Daniel",email: "daniel_m_nascimento@estudante.sesisenai.org.br",authorized: true})
                }, 3000);
            });
        }
        function showResult(res){
            console.log(res)
        }
        getAuth().then(showResult)
