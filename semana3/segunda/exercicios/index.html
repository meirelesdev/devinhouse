Exercicio do dia

  • Livraria

    • Descrição do Desafio

    Criar um sistem ade Cadastro e venda de livros utilizando HTML, CSS, JS. Este Sistema irá conter uma tela de cadastro de livros e uma tela de visualização. A tela de Cadastro de livros conterá um formulário com os seguintes campos:

    • Título do Livro
    • subtítulo
    • Editora
    • Autor
    • Numero de Páginas
    • Preço
    • isbn
    • Ano de Publicação
    • url da imagem de capa

    Esses valores serão salvos numa variável localStorage.
